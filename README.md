# Client Lead Management System (Mini CRM)

## Overview

A Full Stack Lead Management System developed as part of the Future Interns Full Stack Web Development Internship.

## Features

* Add New Leads
* View All Leads
* Delete Leads
* Lead Status Tracking
* Notes Management
* Dashboard Statistics

## Technologies Used

### Frontend

* React.js
* Axios
* CSS

### Backend

* Node.js
* Express.js

## Future Improvements

* MongoDB Database Integration
* User Authentication
* Lead Editing
* Deployment on Vercel and Render
